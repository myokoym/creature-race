# An RPG on browser/desktop by Ruby

An RPG game on browser or desktop (cross platform) by Ruby.

## Powered by

* [DXOpal](https://yhara.github.io/dxopal/doc/ja/index.html) (browser)
* [Gosu](https://www.libgosu.org/) (desktop)

## Usage

### Browser

```
% dxopal server
```

or

```
% rackup
```

### Desktop

```
% ./gosu_main.rb
```
